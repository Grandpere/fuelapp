# SP1-009 - Local authentication baseline (email/password)

## Context
Sprint 1 needs secure auth foundations without external provider dependency.

## Scope
- Implement local user login flow (email/password).
- Add password hashing and secure session setup.
- Add logout and login protection basics.

## Out of scope
- Social/SSO providers.
- Password reset and email verification.

## Acceptance criteria
- User can authenticate locally with secured credentials.
- Invalid credentials are rejected safely.
- Protected API/UI resources require authenticated user.

## Dependencies
- SP1-001.
