# SP19-001 - Front maintenance reminder rules CRUD

## Why
Reminder cards are now actionable when they exist, but front-office users still cannot create or manage the rules that produce them. That leaves the feature incomplete in real usage.

## Scope
- Add front-office create/edit/delete flows for maintenance reminder rules.
- Surface the existing rules directly on the maintenance dashboard.
- Keep admin coverage deferred for now: operations can already inspect reminders, while this ticket is about user ownership of the setup itself.
