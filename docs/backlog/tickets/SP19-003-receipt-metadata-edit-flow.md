# SP19-003 - Receipt metadata edit flow

## Why
Receipt lines are editable, but metadata such as vehicle and station still cannot be corrected easily afterward. That leaves part of the receipt workflow incomplete.

## Scope
- Add a front-office edit flow for receipt metadata.
- Cover at least vehicle, station, odometer, and date if the existing application flow allows it cleanly.
- Keep admin coverage deferred unless operations need a matching recovery path later.
