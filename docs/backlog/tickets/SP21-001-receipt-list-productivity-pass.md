# SP21-001 - Receipt list productivity pass

## Goal
Make `/ui/receipts` faster to operate during daily review and manual entry.

## Scope
- make current filter context easier to read
- add shortcut actions when the list is already scoped to a vehicle
- prefill manual receipt creation from the active vehicle filter
- add a few lightweight date shortcuts instead of forcing repeated manual date entry

## Out of scope
- admin receipt list parity
- repository/schema changes unless strictly required
- replacing the existing list/table structure
