# SP22-001 - Vehicle timeline editing shortcuts

## Why
The vehicle hub is already useful, but recent maintenance items still require extra navigation before a user can correct an event or a plan.

## Expected outcome
- `/ui/vehicles/{id}` exposes direct edit shortcuts on recent maintenance items.
- Actions keep the user anchored to the vehicle page after save.
- The hub reduces the “open maintenance first, then edit” detour for common corrections.

## Notes
- Front-office only for now.
- No admin parity needed at this stage.
