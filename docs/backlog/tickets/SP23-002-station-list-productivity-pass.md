# SP23-002 - Station list productivity pass

## Why
Station detail is now useful, but users still need a front-office list view that surfaces the right shortcuts and basic usage signals before opening each station.

## Expected outcome
- `/ui/stations` exists as a front-office working list.
- Each station row exposes basic usage signals and direct shortcuts into receipts, analytics, and receipt creation.
- The list helps users spot active versus unused stations without opening every detail page.

## Notes
- Front-office only for now.
- No admin parity needed at this stage.
