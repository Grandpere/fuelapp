# SP23-003 - Maintenance dashboard scanability pass

## Why
The maintenance dashboard has become useful, but it now asks the user to read too much before understanding what is urgent, what is coming soon, and what was handled recently.

## Expected outcome
- `/ui/maintenance` highlights urgent reminders, near-term plans, and recent events at a glance.
- Timeline and planner rows expose lightweight scan labels without changing the underlying flows.
- The dashboard reduces the mental load of reviewing multiple reminders, plans, and events in one pass.

## Notes
- Front-office only for now.
- No admin parity needed at this stage.
