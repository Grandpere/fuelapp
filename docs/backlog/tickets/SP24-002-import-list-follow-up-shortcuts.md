# SP24-002 - Import list follow-up shortcuts

## Why
The import detail flow is already much clearer, but the main `/ui/imports` list still forces too many clicks when users just want to keep moving through the next obvious follow-up.

## Expected outcome
- `/ui/imports` exposes quick status filters with counts.
- The page highlights the next import that needs attention without opening each detail page.
- Rows expose secondary shortcuts when users may want both the direct target and the import detail itself.

## Notes
- Front-office only for now.
- No admin parity needed at this stage.
