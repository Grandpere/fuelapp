# SP34-001 - Admin maintenance support toolkit

## Summary
Tighten the admin maintenance toolkit so support can move more quickly between reminders, rules, events, plans, vehicles, and receipts when investigating ambiguous maintenance cases.

## Delivered scope
- add `event_type` filtering on admin reminder investigations
- add vehicle-scoped support toolkit shortcuts on reminder and event lists
- add direct receipts continuity from reminder and event views
- make reminder and event detail pages read more like a support dossier with explicit continuity actions
