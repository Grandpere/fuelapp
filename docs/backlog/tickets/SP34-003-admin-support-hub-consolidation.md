# SP34-003 - Admin support hub consolidation

## Summary
Consolidate the admin dashboard into a clearer support hub so the new investigation tools across imports, maintenance, users, identities, security, and audit are easier to reach from one place.

- expose dedicated entry points for import recovery, maintenance investigation, and account recovery
- keep the dashboard aligned with the newer admin workflows added across recent sprints
