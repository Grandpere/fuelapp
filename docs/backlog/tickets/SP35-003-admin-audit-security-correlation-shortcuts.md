# SP35-003 - Admin audit and security correlation shortcuts

## Summary
Add quicker investigation pivots across admin security and audit flows so actor and correlation-based triage takes fewer hops.

## Delivered scope
- add row-level `Identities` shortcuts whenever an actor-backed user is known
- add row-level `Correlation` shortcuts from security activity into the matching audit trail
- align audit and security list actions around the same investigation workflow
- extend admin functional coverage for the new investigation shortcuts
