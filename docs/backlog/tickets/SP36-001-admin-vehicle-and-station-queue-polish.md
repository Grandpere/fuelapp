# SP36-001 - Admin vehicle and station queue polish

## Summary
Improve the admin vehicle and station queues with clearer scanability, filters, and follow-up shortcuts.

## Delivered scope
- add queue-level filters and active-filter summaries to the admin vehicle and station lists
- add `Signal` columns so support can scan the next useful vehicle or station without opening every row
- add queue shortcuts for the next vehicle or station likely to need follow-up
- extend admin functional coverage for the touched queue contracts
