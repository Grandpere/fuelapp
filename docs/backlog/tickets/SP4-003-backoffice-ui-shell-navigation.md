# SP4-003 - Back-office UI shell and navigation

## Context
Operational workflows need a dedicated UI structure.

## Scope
- Create back-office layout (sidebar/top nav).
- Add list/detail pages for core entities.
- Keep responsive behavior for desktop/tablet.

## Out of scope
- Advanced visual theme system.

## Acceptance criteria
- Admin can navigate entities/jobs from one entrypoint.
- UI works on common viewport sizes.

## Dependencies
- SP4-001, SP4-002.
