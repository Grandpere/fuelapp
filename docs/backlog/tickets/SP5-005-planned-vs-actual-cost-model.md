# SP5-005 - Planned vs actual maintenance cost model

## Context
Forecasting maintenance budget is a core value add.

## Scope
- Add planned maintenance cost entries.
- Link actual events to planned items when relevant.
- Compute variance metrics.

## Out of scope
- Advanced financial reporting.

## Acceptance criteria
- Planned and actual costs are comparable per period/vehicle.
- Variance is queryable.

## Dependencies
- SP5-002.
