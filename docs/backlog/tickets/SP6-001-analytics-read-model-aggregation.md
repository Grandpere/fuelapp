# SP6-001 - Analytics read model and aggregation strategy

## Context
Dashboards should avoid heavy runtime queries.

## Scope
- Define analytics read model and aggregation jobs/views.
- Establish refresh strategy (sync/async).
- Track freshness metadata.

## Out of scope
- Final dashboard UX.

## Acceptance criteria
- KPI queries are predictable and efficient.
- Aggregation pipeline is observable.

## Dependencies
- Prior data domains stable.
