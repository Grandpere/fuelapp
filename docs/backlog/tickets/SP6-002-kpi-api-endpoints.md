# SP6-002 - KPI API endpoints

## Context
UI and external clients need consistent KPI access.

## Scope
- Add endpoints for key metrics (cost/month, avg price/L, consumption).
- Support period and vehicle filters.
- Return explicit units/precision.

## Out of scope
- Advanced charting config.

## Acceptance criteria
- KPI responses are stable, documented and tested.
- Filtered responses match source data.

## Dependencies
- SP6-001.
